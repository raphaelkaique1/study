# Wireshark
Trata-se de um analisador de protocolos opensource utilizado para verificar o funcionamento de uma rede em detalhes e, com isso, entre outras coisas como:
- identificar falhas de hardware
- identificar fonte de ataques DoS
- localizar programas mal-intencionados
- auxiliar desenvolvedores
- estudo da arquitetura de uma rede

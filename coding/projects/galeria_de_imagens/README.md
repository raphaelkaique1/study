# Galeria de Imagens

Este é um projeto simples de uma galeria de imagens, desenvolvido com HTML e CSS. A galeria exibe imagens em um formato de carrossel horizontal, com efeitos de hover para aumentar a imagem e aplicar sombras.

## Funcionalidades
- Exibição de várias imagens em uma linha.
- Efeito de zoom nas imagens ao passar o mouse (hover).
- Layout responsivo, ajustando-se ao tamanho da tela do dispositivo.
